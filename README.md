# SQL Starter HIC

This project is used to develop and validate iframe snippet code that can be embedded into Jira and Confluence pages.

## Purpose

- Build reusable HTML snippets for iframe-based integrations.
- Provide quick examples for embedding and testing content.
- Support publishing and review workflows via static HTML pages.

## Project contents

- `snippet_000.html` through `snippet_013.html`: iframe snippet variants.
- `getting-started-draft.html`: draft getting-started page.
- `index.html`: a central index page to open GitHub and navigate all HTML pages in this repository.
- `generate_snippets.py`: helper script for snippet generation/maintenance.
